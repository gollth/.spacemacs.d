# Spacemacs Config
My general customization for Spacemacs

## Setup
```
$ git clone http://github.com/gollth/.spacemacs.d ~/.spacemacs.d
$ ln -s ~/.spacemacs.d/.spacemacs ~/.spacemacs
```
